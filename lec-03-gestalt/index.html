Gestalt-принципы

  • Между создателем визуализации и зрителем нет известного соглашения по порядку просмотра (ср. печатные тексты)
  • \(\implies\) Нужно делать визуализации так, чтобы направлять внимание зрителя, исходя из общих законов человеческого восприятия
  • Gestalt-принципы — универсальные принципы цельности, «разумности» восприятия
  • Зная Gestalt, можно убирать лишние элементы с изображений, облегчая восприятие и не боясь, что без «визуальных костылей» зритель что-то воспримет не так

Упражнения

  • Постройте свои цветовые палитры для различных видов данных с помощью инструмента ColorPicker
  • Подумайте, какие Gestalt-принципы “сильнее”, какие “слабее”: если объекты на картинке одновременно могут быть сгруппированы по цвету на основе similarity, или по близости на основе proximity, то какая именно группировка приходит в голову первой? А для других комбинаций пар gestalt-принципов?
  • Какой (какие) gestalt-принципы эксплуатируются в инструменте на этом видео?
  • Найдите в интернете несколько графиков и подумайте, как, используя gestalt-принципы, можно было бы упростить графики и/или убрать из них “лишние пиксели”, чтобы увеличить пропагандируемый Эдвардом Тафти data-ink ratio.
